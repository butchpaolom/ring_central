To run server:
1. Install python (https://www.python.org/).
2. Open console inside project folder and type the command 'pip install -r requirements.txt', make sure you are in the right directory (ring_central/crud-prac/crud). 
3. After installing packages, navigate inside crud folder (cd crud).
4. Run the command in console 'python manage.py runserver'.
5. It's gonna show where the development server is running, by default it is localhost:8000
